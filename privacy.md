# Privacy Policy

Resize does not collect, store, transmit, or share any personal data.

The extension saves your most recent custom width and height values locally using Chrome's session storage, so the input fields stay filled while you work. This data never leaves your device and is automatically cleared when you close your browser.

The extension also stores a short per-window undo/redo history of window dimensions and position in Chrome's session storage so you can revert recent resizes. Like the custom values, this is local-only and cleared when the browser closes.

Resize does not use analytics, trackers, cookies, or any third-party services. It does not read or store the content of any web page you visit. When you trigger a resize, the extension reads only the current tab's viewport dimensions (width and height in pixels) to calculate the correct window size, and this information is used immediately and never stored.

If you have questions, open an issue at https://github.com/bryanrea/Resize
